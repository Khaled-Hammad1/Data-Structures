# Data Structures Projects in C

This repository contains four projects implemented in **C language** for the **Data Structures and Algorithms** course.  
Each project focuses on applying a specific data structure or algorithm to solve a practical problem.

## Projects Overview

### 1. Linked List and Radix Sort
This project implements a two-level hierarchy for Palestine districts and towns using a **doubly linked list**.  
It sorts districts alphabetically using **Radix Sort** and sorts towns within each district based on population.

**Main topics:**
- Doubly Linked List
- Radix Sort
- File handling
- Sorting hierarchical data

---

### 2. Text Editor with Undo/Redo
This project implements a text editor using **two stacks** and **one queue**.  
It supports loading text from a file, inserting and removing strings, and performing **Undo/Redo** operations.

**Main topics:**
- Stack using Linked List
- Queue
- Undo/Redo mechanism
- String manipulation
- File I/O

---

### 3. AVL Tree and Hash Table Word Indexer
This project reads words from a text file, stores them in an **AVL tree** with their frequencies, then creates a **hash table** for fast access.  
It also supports searching, inserting, deleting, and displaying word statistics.

**Main topics:**
- AVL Tree
- Hash Table
- Word frequency counting
- Search and deletion
- Data indexing

---

### 4. Course Prerequisite Planner and Campus Route Finder
This project implements two applications:
1. Finding the shortest route between campus buildings using **Dijkstra's Algorithm**
2. Sorting courses based on prerequisites using **Topological Sort**

**Main topics:**
- Graph representation
- Dijkstra Algorithm
- Topological Sort
- File handling
- Course planning

---

## Technologies Used
- **Language:** C
- **Concepts:** Linked Lists, Stacks, Queues, AVL Trees, Hash Tables, Graphs, Sorting Algorithms

## Purpose
The purpose of this repository is to practice core data structures and algorithms by building complete console-based applications in C.
